# QuotaBar v1.0.0

首个正式版本，集中展示 OpenCode Go、Kimi Code、Codex 和 DeepSeek 的额度或余额状态。

## 安装

- 下载 `QuotaBar-1.0.0.dmg`，打开后将 QuotaBar 拖入 Applications；或下载 ZIP 解压后移动应用。
- 下载前后可用 `SHA256SUMS.txt` 校验文件完整性。
- 应用已使用 Developer ID 签名并经 Apple 公证，需要 macOS 14 或更高版本。

## 隐私与权限

- DeepSeek API Key 和手动 Kimi Token 保存在系统 Keychain。
- Kimi 浏览器 Cookie 导入仅在用户启用完全磁盘访问权限时工作。
- QuotaBar 不记录 Token、Cookie、Authorization Header 或原始私密响应。

## 已知边界

- OpenCode Go 用量通过官方控制台查看，不要求安装 OpenCode CLI。
- Kimi 月额度当前提供订阅页入口，尚无经过真实账号验证的稳定机器查询接口。
